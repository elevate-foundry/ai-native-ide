🔒 Privacy & Data Collection

Aria uses BBID (Behavioral Biometrics) to create a unique device fingerprint. This includes:

  • Device characteristics (screen, browser, OS)
  • Behavioral patterns (mouse, keyboard, scroll)
  • Canvas/WebGL fingerprints

This data is stored locally only and never sent to external servers.

View detailed privacy policy

🔒 Privacy Policy

What We Collect

  • Device Info: Screen resolution, color depth, timezone, language, browser
  • Hardware: CPU cores, memory, touch support
  • Canvas Fingerprint: Unique rendering characteristics
  • WebGL Fingerprint: GPU vendor and renderer
  • Behavioral Data: Mouse movements, keystroke timing, scroll patterns

How We Use It

BBID creates a unique identifier for your device to demonstrate behavioral biometrics technology. This is used solely for demonstration purposes within this application.

Data Storage

All data is stored locally in your browser using localStorage. No data is transmitted to external servers.

Your Rights

  • Access: View all collected data via the "View My Data" button
  • Erasure: Delete all data via the "Delete My Data" button
  • Withdraw Consent: Disable BBID at any time in Settings

Data Retention

Data is retained until you delete it or clear your browser storage.
